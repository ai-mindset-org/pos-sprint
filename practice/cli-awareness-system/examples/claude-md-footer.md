# Шаблон: Footer Protocol для CLAUDE.md

> Скопируй этот блок в свой CLAUDE.md (глобальный или проектный).
> Адаптируй под свои нужды — убери лишнее, добавь нужное.

---

## Context Management Protocol

### Footer (в каждом содержательном ответе)

**Формат:**
```
◇ ctx: {used}K | {left}K left | {pct}% | {symbol}
```

**Символы режима:**
- `●` — OK (< 50%) — полный функционал
- `◐` — WARN (50–70%) — компактные ответы, экономия контекста
- `○` — CRIT (> 70%) — только handoff, не начинать новые задачи

**Откуда брать данные:** Status Line Claude Code показывает `▰▰▰▱▱ XX%` — использовать эти значения.

### Поведение по порогам

| Порог | Режим | Что делать |
|-------|-------|------------|
| < 50% | ● OK | Полный функционал |
| 50–70% | ◐ WARN | Компактные ответы, экономить контекст |
| > 70% | ○ CRIT | Только handoff — сохранить прогресс, передать контекст |

### Handoff при CRIT (> 70%)

Генерировать блок передачи:

```markdown
---
## Handoff

**Проект:** [название]
**Путь:** `[путь]`

### Сделано
- [что было сделано]

### Осталось
1. [ ] Задача 1
2. [ ] Задача 2

### Prompt для продолжения
Продолжи работу над [проект]...
---
```
